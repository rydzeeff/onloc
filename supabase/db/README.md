# Supabase DB schema for Codex (documentation snapshot)

## Что это
Schema-only снимок структуры БД для анализа проекта агентом (Codex) и разработчиками.

Файлы:
- `public_schema.sql` — схема приложения (`public`)
- `external_tables_schema.sql` — выборочно важные таблицы Supabase (`auth.users`, `storage.*`), если используются в коде

## Важно
- Здесь **нет данных** (rows), только структура.
- Не хранить в этой папке дампы с данными.
- Перед коммитом проверять, что в SQL-файлах нет чувствительных литералов/секретов.

## Источник
Self-hosted Supabase DB на сервере `192.168.3.24` (контейнер `supabase-db`).

## Как обновлять
Запускать команды schema dump с фронт-сервера (`192.168.3.23`) и сохранять файлы в этот каталог.
